models/

## 📤 Comment ajouter les fichiers

1. Place tes fichiers `.3mf` dans ce dossier `models/animals/`
2. Nomme-les comme en haut (ex: `vache-herens.3mf`)
3. Je vais les convertir en GLB et les intégrer dans la landing page

## 🔄 Conversion

Les fichiers 3MF seront :
- Convertis en GLB (optimisé pour le web)
- Affichés en rotation 3D sur la page `/entreprises`
- Captures d'écran haute qualité générées automatiquement
